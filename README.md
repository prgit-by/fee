PROIT Fee
=========

## Installation ##

Need use Python version more than 3.4.

Steps:

* sudo apt-get install libxml2-dev libxslt1-dev
* pip3 install $(cat ./requires.txt)

## Execution ##

    python3 -m fee


## ENV variables ##

To see files .env

    PYTHON_BIN_FILE - 
    PROIT_FEE_URL - 
    PROIT_FEE_KEY - 
    PROIT_FEE_YEAR - 
    PROIT_FEE_PERCENT - 
    PROIT_FEE_FILE -
